#IBM HR Analytics Dashboard
Overview
The IBM HR Analytics Dashboard is an interactive Tableau project that provides insights into workforce demographics, employee attrition. It enables HR professionals and business leaders to identify trends, understand employee behavior, and make informed, data-driven decisions.
